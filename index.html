Home network upgrade project

Background

My home network system in China was a 100Megabit(100Mbps) base with wireless supported network environment, after upgraded to 120Mbps network from ISP the old network devices is not able handle it anymore, so I decided to upgraded the whole network system during summer break.

Problems

  • Needs more bandwidth to handle LAN traffic
  • WiFi is not able to cover the entire house
  • Hard to troubleshooting when I am not in China
  • My family looks for more wire connection

Solution

First thing first, upgraded the switch to fully gigabit and POE+ so it's able to handle the traffic, and then I used two FIT Access Point to cover the entire house and control by the wireless controller(WLC), the reason I use WLC is because it's allow me to config and monitor both ap and AP roaming. Also connect few new Cat6 cable to my family's bathroom so they prefer wire. And finally I setup a SSLVPN so I can remotely control my network anywhere in the world if there is internet.

Devices List

# Name Brand For
1 H3C SecPath F100-C-G2 H3C Core VPN Firewall
2 Cisco C2960L-16PS-LL Cisco Backbone Switch
3 Cisco AIR-CT2504-5-K9 Cisco Wireless Controller
4 Cisco AIR-AP1852I-H-K9 Cisco Access Point

Those may looks overkill to a home network, but let me show you some advantages about it.

  1. Fully gigabit network allow more services available such as VoIP and CCTV system
  2. H3C firewall provided VPN for remote access, not just SSL but more
  3. FIT AP and WLC allow truly seamless roaming without loss connection when switch between AP
  4. Stable, important than anything
